The Internet Protocols

  1. TCP - The Transmission Control Protocol
  2. UDP - The User Datagram Protocol

The Transmission Control Protocol

Once your computer connects to the host server, the server follows transmission control protocol. Devices use TCP when all the requested information needs to be transferred.

For example:
  • website
  • picture
  • video

Process:
  1. the client computer sends request to connect to the server
  2. server accepts the request
  3. the client requests specific packets
  4. the server sends all the requested packets
  5. the client checks whether all the requested packets were delivered
  6. if they weren't delivered, the client requests those sepcific packets again
  7. after all the packets were sent and received, the connection is closed

The User Datagram Protocol

User datagram protocol is another way to transfer information. It focuses more on speed than on getting every packet to the final destination. This protocol is used when speed is more important tahn getting all the requested information.

For example:
  • video conferencing
  • online games
  • media streaming
